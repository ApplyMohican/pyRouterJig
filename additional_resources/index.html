Additional Resources

Tutorials

I highly recommend the following tutorials put together by Phil Barrett:

  • Custom Box Joints using the Incra LS System. This is an excellent introduction to pyRouterJig.
  • Box Joint Clamping Cauls using pyRouterJig.

Phil makes beautiful boxes and other wood projects. He has worthwhile tips not only for pyRouterJig, but also for woodworking and router use in general. Take a look at his portfolio.

Websites

  • http://www.routerforums.com/ is an excellent forum on routers, with very nice and patient members. No question is too stupid for them, as I have demonstrated many times.
  • http://www.wood-database.com/ is an excellent place to get wood grain images. See the documentation Wood Pattern Selection on how to use these images.
